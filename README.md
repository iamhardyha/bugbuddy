# bugbuddy
🐞 BugBuddy — 개발자들이 막히는 문제를 함께 해결하는 Q&amp;A + 1:1 멘토링 플랫폼

## 🧰 기술 스택 (Tech Stack)

> 본 프로젝트의 기술 스택은 MVP 단계 기준으로 구성되어 있으며,  
> 서비스가 성장함에 따라 점진적으로 확장 및 업데이트될 예정입니다.

### Frontend
- **Next.js (React)**: 웹 서비스 UI 및 라우팅
- **TypeScript**: 타입 안정성 및 유지보수성 향상

### Backend
- **Spring Boot (Kotlin)**: REST API 서버

### Database
- **MySQL 8**: 서비스 데이터 저장

### Infra / DevOps
- **Docker & Docker Compose**: 로컬 개발 환경 구성
